Preguntas y Respuestas sobre CSS y HTML

¿Qué es CSS?
CSS significa "Cascading Style Sheets" y se utiliza para darle estilo a la pagina (bueno a sus elementos).

¿Cuál es el propósito de CSS?
El propósito de CSS es darle estilo visual a la página web como colores, tamaños, márgenes, etc.

¿Cómo se aplica un CSS a un HTML?
Tienes que crear un apartado "Style" donde iras aplicando diversos comandos que modificaran
cada uno de los elementos.

¿Qué es un selector?
Un selector en CSS es lo que le dice al navegador qué elemento HTML debe ser estilizado.

¿Qué es una regla CSS?
Debe ser aplicado en un apartado "style>" ya sea en el mismo documento o en uno aparte

¿Qué es un selector de tipo de etiqueta?
Es un selector que selecciona todos los elementos de una etiqueta específica. Ejemplo: p aplica a todos los párrafos.

¿Qué es un selector de clase?
Un selector de clase se aplica a todos los elementos que tienen una clase específica. Ejemplo: .miClase

¿Qué es un selector de id?
Un selector de id se aplica a un solo elemento con un id específico. Ejemplo: #miId

¿Qué es el modelo de caja CSS?
El modelo de caja define cómo se estructuran los elementos HTML. Incluye: contenido, relleno, borde y margen.

¿Cómo cambiar el color de la fuente con CSS?
Se usa la propiedad color en CSS. Ejemplo: p { color: blue; }

HTML Básico

¿Qué es HTML?
HTML significa "HyperText Markup Language" y es el lenguaje usado para estructurar contenido en la web.

¿Qué son las etiquetas y ?
La etiqueta contiene metadatos, y la etiqueta contiene el contenido visible de la página.

¿Qué son las etiquetas section, article, aside, header, footer?
Son etiquetas semánticas que estructuran el contenido: section es una sección general, article es un contenido autónomo, aside es contenido adicional, header es el encabezado, y footer es el pie de página.
